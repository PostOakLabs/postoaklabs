Agentic Payments AP2 · ISO 20022 AP2 Policy JSON

A visual guardrail for machine-readable
payment agent mandates.

Paste an Policy Mandate JSON mandate on the left. The tool parses every constraint, renders it as a shield-gated flow chart through the agent's decision pipeline, and compiles a validated mandate ready to hand off to your MCP agent runtime. Every shield is a contract: the agent must satisfy all of them before settling a payment.

Value at risk · unscoped agent transactions

Full transaction loss + unauthorized-debit liability per breach

Autonomous agents executing payments without a client-side policy guard can fall outside the principal's intent envelope — wrong merchant, wrong amount cap, or an expired window. Policy Mandate v1 policy JSON enforces the envelope before the MCP tool call resolves, so the agent cannot emit a payment that the principal did not authorize. Visualizing the mandate before deployment is the cheapest path to catching scope-drift.
