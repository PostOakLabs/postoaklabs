From prototype to platform

The original 27 tools on this page were proof-of-concept prototypes built to test the AINumbers thesis: that structured fintech intelligence, delivered client-side with zero PII, could be genuinely useful to practitioners.

They proved it. The catalog grew to 268 tools across payments, compliance, open banking, stablecoin, ISO 20022, CFPB §1033, Basel IV, and more — each with an AP2 v1.0 policy mandate export and a live MCP manifest for agentic retrieval.

For a curated, demo-first experience, start with the Showcase. For the full catalog with MCP integration, visit ainumbers.co.
